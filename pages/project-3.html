Retail Order Builder

Full-stack technical test built with Spring Boot and Angular that lets users search products, build an order, and calculate totals with VAT in real time.

Contact for Demo

Project Overview

This project simulates a retail workflow: search for products, add them to an order, adjust quantities, and submit the order to a backend API. The UI shows net, VAT (21%), and total values, while the backend persists a consistent snapshot of prices and totals.

The backend exposes product and order endpoints, runs on an in-memory H2 database, and includes validation and integration tests. CORS is configured so the Angular frontend can call the API locally.

Key Features

Spring Boot API
Angular order UI
Product search
VAT calculation (21%)
H2 in-memory DB
Tests and validations

Local Setup

Start the backend with Maven, then run the Angular frontend. Once both are up, the app is available at localhost:4200 and the API at localhost:8080.
